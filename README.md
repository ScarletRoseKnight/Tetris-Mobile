# ☢️ Tetris-Mobile: Fallout 4 Pip-Boy Edition

A retro-futuristic Tetris clone for Android, meticulously designed to replicate the terminal aesthetic of the **Fallout 4 Pip-Boy**. Built with Kotlin and custom SurfaceView rendering.

## 🟢 Features
- **Pip-Boy 3000 Interface**: Monochrome "Phosphor Green" visuals with custom CRT scanline overlays.
- **Dynamic Glow System**: Real-time phosphor glow and shadow layers for a truly radioactive look.
- **Terminal Logic**: Stylized start menu, "Vault-Tec" branding, and "Fatal Error" game-over screens.
- **Persistent Data**: High scores are automatically saved to the device terminal (SharedPreferences).
- **"How to Play" Guide**: Built-in instruction terminal accessible from the main menu.

## 📱 Mobile Controls
The game is optimized for one-handed mobile play with intuitive touch gestures:
- **Tap**: Rotate piece clockwise.
- **Swipe Left / Right**: Move piece.
- **Swipe Down**: Soft drop for faster placement.
- **Swipe Up**: Alternate rotation gesture.
- **Start / Restart**: Tap the dedicated terminal buttons or the screen to reboot.

## 🕹️ Gameplay Mechanics
- **Full Tetromino Set**: Features all seven classic shapes (I, J, L, O, S, T, Z).
- **Scoring System**:
  - 1 Line: 100 points
  - 2 Lines: 300 points
  - 3 Lines: 500 points
  - 4 Lines (Tetris): 800 points
- **High Score Tracking**: Your "BEST RECORD" is displayed on both the gameplay and game-over screens.
